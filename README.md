# Climate Action Hackathon 2025
### *Urban Heat & Flood Resilience (SDG 13: Climate Action)*

A clean and responsive **landing page** promoting the Climate Action Hackathon 2025 — focused on **urban heat** and **flood resilience**.  
Built using **HTML, CSS**, it highlights event goals, schedules, speakers, and registration details with an **SDG 13 green theme**.

---

## Overview

- **Purpose:** Encourage climate innovation and awareness under SDG 13  
- **Tech Stack:** HTML5 · CSS
- **Theme Color:** SDG 13 Green (#3F7E44)  
- **Responsive Design:** Works on all devices  
- **Sections:** About · Schedule · Speakers · Tracks · Registration · Contact  

---

## Key Features

 Sticky & responsive navigation  
 Hero banner with call-to-action  
 Speaker cards with hover animation  
 Structured schedule layout  
 Smooth scrolling and accessibility  
 Lightweight and easy to customize
